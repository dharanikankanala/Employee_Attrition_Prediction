# Employee Attrition Prediction

## Project Overview

This project predicts employee attrition using Machine Learning techniques on the IBM HR Analytics Employee Attrition dataset.

## Objectives

* Analyze employee attrition patterns
* Identify factors influencing employee turnover
* Build predictive models
* Generate HR recommendations

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Google Colab

## Models Implemented

1. Logistic Regression
2. Random Forest Classifier
3. Gradient Boosting Classifier

## Evaluation Metrics

* Precision
* Recall
* F1 Score
* ROC-AUC Score
* Confusion Matrix

## Key Findings

* Overtime is a major contributor to attrition.
* Lower job satisfaction increases employee exit risk.
* Sales Representatives show the highest attrition rates.
* Employees in their first few years are more likely to leave.
